I built this project to practice Python GUI programming and learn how to display real-time data.

-> Digital Clock using Python:

This project is a real-time digital clock built using Python and Tkinter.

-> Features:

- Displays current time in HH:MM:SS format
- Shows current date below the time
- Updates automatically every second
- Simple and user-friendly GUI


-> What I learned:
- Working with Tkinter widgets like Label
- Formatting time & date with datetime module
- Using event-driven programming with 'after()' method


-> Technologies Used:
- Python
- Tkinter
- datetime module

-> How to Run:
1. Install Python
2. Run the file:
   python digital_clock.py
